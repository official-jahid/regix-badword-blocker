✦ Features

Everything You Need to Moderate

REGIX combines AI-powered content filtering with a robust penalty system, giving you complete control over your Discord server's chat environment.

🤖

AI-Powered Detection

Smart content analysis pipeline that detects bad words, spam patterns, and inappropriate content with high accuracy. Supports custom blacklists and whitelists.

🔨

Strike & Penalty System

Configurable strike-based enforcement. Automatically warns, times out, or bans users based on violation severity and strike thresholds. Fully customizable actions.

🌐

Full-Web Dashboard

Beautiful dark-themed control panel with real-time stats, user management, activity logs, and bot configuration. JWT-authenticated with role-based access control.

⚙️

Real-Time Configuration

Add or remove bad words, manage whitelist, configure ignored channels, and adjust bot settings on the fly without restarting the bot — all through the dashboard.

📊

Activity Logging

Comprehensive audit logging with every action tracked. Export logs as CSV or JSON, filter by user or action type, and review moderation history.

🛡️

Role-Based Access

Three-tier access system: Owner, Admin, and Mod. Granular permissions for each command and dashboard feature. Secure JWT authentication with token refresh.
